What is CS Coin?
--------------

[CS Coin](https://cscoin.io/) (formerly known as Zel) is a fork of 2.1.0-1 Zcash aiming to provide a decentralized development platform via CS Nodes and more.

CS Coin is PoW and "ASIC resistant" with ZelHash (Modified Equihash 125,4) with the personalisation string ZelProof and utilises the LWMA3 difficulty algorithm.

## :rocket: Getting Started

Please see our [wiki](https://wiki.cscoin.io/) for any and all info.

To setup a CS Node please follow this [guide](https://medium.com/@mmalik4/cs coin-light-node-setup-as-easy-as-it-gets-833f17c73dbb) and this [video guide](https://www.youtube.com/watch?v=KYWUXrKP9do).

### Building

CS Coin Daemon build and installation guide is [here](https://zel.gitbook.io/zelcurrency/installing-zel-daemon).

If you have the dependencies you can build CS Coin Daemon from source by running:

```
./zcutil/build.sh -j$(nproc)
```
